# GEMINI.md

## 프로젝트 개요

이 프로젝트는 가상의 글로벌 식품 회사인 "농신"의 정적 웹사이트입니다. 현대적인 단일 페이지 프로모션 웹사이트로 보입니다.

**사용된 기술:**

*   **HTML5**
*   **CSS3**
*   **JavaScript (ES6)**
*   **외부 라이브러리:**
    *   [Swiper.js](https://swiperjs.com/): 터치 지원 슬라이더 및 캐러셀을 만들기 위해 사용됩니다.
    *   [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/): 스크롤 시 발생하는 애니메이션을 위해 사용됩니다.
    *   [Google Fonts](https://fonts.google.com/): 'Noto Sans KR' 글꼴을 사용합니다.

**아키텍처:**

*   **`index.html`**: 전체 웹페이지의 구조를 정의하는 단일 HTML 파일입니다. 헤더, 메인 콘텐츠 섹션, 빠른 접근 사이드 메뉴를 포함합니다.
*   **`style.css`**: 레이아웃, 색상, 타이포그래피, 애니메이션을 포함한 웹사이트의 모든 시각적 규칙을 담고 있는 주요 스타일시트입니다.
*   **`script.js`**: 외부 라이브러리(Swiper, AOS)를 초기화하고, 스크롤-투-탑 버튼 및 스크롤 시 헤더 효과와 같은 사용자 정의 동작을 구현하는 JavaScript 로직을 포함합니다.
*   **`media.css`**: 이 파일은 존재하지만 현재 비어 있습니다. 반응형 디자인 재정의를 위해 만들어진 것으로 보입니다.
*   **`image/`**: 이 디렉토리는 웹사이트에서 사용되는 모든 이미지를 저장하며, 배경, 아이콘, 로고를 위한 하위 디렉토리로 구성되어 있습니다.

## 빌드 및 실행

이것은 빌드 과정이 없는 정적 웹사이트입니다. 프로젝트를 실행하려면 웹 브라우저에서 `index.html` 파일을 열기만 하면 됩니다.

라이브 서버를 모방하고 로컬 리소스를 가져올 때 발생할 수 있는 CORS 문제를 처리하는 더 나은 개발 경험을 위해 간단한 로컬 웹 서버를 사용할 수 있습니다.

**Python 사용:**
```bash
# 프로젝트의 루트 디렉토리에 있는지 확인하세요
python -m http.server
```
그런 다음 브라우저에서 `http://localhost:8000`을 엽니다.

**Node.js 사용 (`serve` 패키지):**
```bash
# 아직 설치하지 않았다면 먼저 serve 패키지를 전역으로 설치하세요
npm install -g serve

# 프로젝트의 루트 디렉토리에서 서버를 실행하세요
serve .
```
그런 다음 `serve` 명령이 제공하는 URL을 브라우저에서 엽니다.

## 개발 컨벤션

*   **스타일링**: 이 프로젝트는 모든 스타일에 단일 `style.css` 파일을 사용합니다. Sass나 Less와 같은 전처리기는 사용하지 않습니다. 스타일은 페이지 섹션별로 구성됩니다.
*   **JavaScript**: JavaScript는 동적 동작을 추가하고 타사 라이브러리를 초기화하는 데 사용됩니다. 코드는 `script.js`에 있으며 `index.html`의 `<body>` 태그 하단에 포함됩니다.
*   **의존성**: CSS 및 JavaScript 라이브러리는 `index.html` 파일에서 CDN을 통해 직접 연결됩니다.
*   **반응형 디자인**: `style.css` 파일에는 반응형을 위한 일부 미디어 쿼리가 포함되어 있으며, `viewport` 메타 태그는 HTML 헤드에 설정되어 있습니다. `media.css` 파일은 사용되지 않습니다.
*   **코드 구조**: HTML은 적절한 경우 시맨틱 태그(예: `<header>`, `<main>`, `<section>`)를 사용하여 잘 구조화되어 있습니다.
